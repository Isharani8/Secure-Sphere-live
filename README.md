# 🛡️ SecureSphere Live

SecureSphere Live is an AI-powered Cyber Threat Intelligence, Awareness, and Incident Response Platform designed to help users understand cyber threats, learn cybersecurity concepts, and access trusted security resources in one place.

## 🚀 Features (Planned)

- 🌍 Live Cyber Threat Dashboard
- 🤖 AI Cybersecurity Assistant
- 📚 Attack Encyclopedia
- 📰 Cybersecurity News
- 🚨 Incident Response Guides
- 🧰 Security Tools
- 🎓 Learning Hub

## 🛠️ Tech Stack

- React + Vite
- Tailwind CSS
- Spring Boot
- Google Gemini API
- Chart.js
- Leaflet

## 📌 Status

🚧 Currently under active development.
